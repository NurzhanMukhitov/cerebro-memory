## Как обновлять этот файл

- Редактируй `core/user-profile.md` в репо на Mac, делай commit и push.
- На VPS: `cd ~/cerebro-memory && git pull`. Файл уже привязан к `~/.openclaw/workspace/USER.md` симлинком, после pull бот увидит изменения после перезапуска: `systemctl --user restart openclaw-gateway`.
- Либо дополняй факты в диалоге и подтверждай (Commit / Policy Lock) — тогда они попадут в память сессии/Strategic Memory, а сюда можно перенести вручную, если хочешь держать один источник истины.
