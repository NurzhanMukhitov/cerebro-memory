# Чат / топик «Tech»

## Назначение

Техническое управление системой Cerebro: настройка системы, диагностика ошибок, управление интеграциями, установка и обновление skills, работа с сервером.

Tech домен обслуживает инфраструктуру Personal AI OS.

## Агент домена

**Primary Agent:** Tech Advisor

Tech Advisor отвечает за: диагностику системы, настройку интеграций, установку навыков, управление деплоем.

## Маршрутизация

Топик **Tech** → **Primary Agent:** Tech Advisor, **Tag:** tech-thread

Chief of Staff может подключаться для координации.

## Типичные запросы

- Всё ли работает в системе?
- Почему бот не видит календарь?
- Как установить новый skill?
- Как обновить OpenClaw?
- Как проверить интеграцию Strava?

## Обязательные источники данных

- deploy/README.md
- protocols/*
- gateway logs
- system status

## Основные области управления

| Область | Описание |
|---|---|
| Gateway | обработка сообщений |
| Skills | подключаемые функции |
| Integrations | внешние сервисы (Telegram, Apple Health, Strava, Calendar) |
| Deployment | обновление системы |
| Security | безопасность системы |

Перед установкой skill: проверка источника, безопасности, совместимости (protocols/skills-integration.md, protocols/security).

## Память

Tech может записывать: logs/system-events.log, docs/system-config.md

## Ограничения

Tech Advisor не выполняет опасные команды без подтверждения, не удаляет данные без разрешения, не изменяет архитектуру системы без согласования.

## Конфликт рекомендаций

Tech имеет самый низкий приоритет. Tech не вмешивается в решения пользователя.

## Итог

Tech Domain — инфраструктурный домен, обеспечивает стабильность системы и работу интеграций.
