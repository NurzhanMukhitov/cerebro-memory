# Health

Папка для заметок по здоровью: сон, стресс, питание, восстановление, профилактика, трекинг показателей.

Наполнять по желанию. Health Advisor и Chief of Staff могут использовать контент при рекомендациях (в рамках протоколов: не диагнозы, при тревожных симптомах — направление к врачу).

## Источники данных

Для рекомендаций по здоровью и спорту Cerebro использует несколько типов данных:

- `data/apple-health-snapshot.md` — свежие 7 дней из Apple Health (сон, шаги, пульс/HRV, вес, температура тела и др.).
- `data/apple-health-baseline.md` — более длинная история Health (до ~90 дней) с агрегацией по дням.
- `data/strava-snapshot.md` — снимок активности из Strava (тип тренировки, длительность, объём за неделю/месяц).
- `health/log-YYYY-MM-DD.md` — ручные ежедневные логи пользователя: питание, самочувствие, тренировки, заметки.
- Текущее время и дата — через tool `session_status` (локальное время Europe/Madrid).

При анализе состояния и планировании нагрузок Health Advisor и Sport Advisor должны по возможности опираться на комбинацию этих источников, а не на один отдельный сигнал.

## Структура журналов

Рекомендованный формат для ежедневных логов (ручных и по напоминаниям из Telegram):

- файлы вида `health/log-YYYY-MM-DD.md`, например:
  - `health/log-2026-03-03.md`
  - `health/log-2026-03-04.md`

Внутри — краткий дневник по дню, например:

```markdown
# 2026-03-03 — Health log

## Статус
- OK / Sick / Recovering / Overreached? (краткий статус дня по здоровью и нагрузке).

## Сон
- Лёг в 00:30, встал в 08:00 (~7.5 ч), просыпался 1 раз.

## Питание
- Завтрак: ...
- Обед: ...
- Ужин: ...
- Перекусы: ...

## Тренировки / активность
- Тренировка: ...
- Шаги / прогулки: ...

## Самочувствие / заметки
- Уровень энергии, стресс, симптомы и т.п.
```

Агент может использовать эти файлы вместе с данными Apple Health / Strava / Ultrahuman для анализа состояния и рекомендаций Health/Sport.

### Статус дня

Статус дня — это краткая оценка состояния здоровья и нагрузки в конкретную дату. Рекомендуемые значения:

- `OK` — всё нормально, можно следовать обычному плану тренировок и дел.
- `Sick` — болею (простуда, температура, острое недомогание).
- `Recovering` — восстановление после болезни или заметного ухудшения самочувствия.
- `Overreached?` — подозрение на перегруз (много тренек + плохой сон/HRV, сильная усталость).

Статус фиксируется в секции `## Статус` файла `health/log-YYYY-MM-DD.md` одной-двумя строками. Health Advisor и Sport Advisor должны учитывать этот статус при любых рекомендациях по тренировкам, нагрузке и режиму дня.
