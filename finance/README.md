# Finance

Папка для заметок по финансам: бюджет, подушка, долги, накопления, крупные решения, налоговый контекст.

Наполнять по желанию. Finance Advisor и Chief of Staff могут опираться на контент при рекомендациях. Решения с серьёзными налоговыми/юридическими последствиями — к профильному специалисту (gestor, юрист).
