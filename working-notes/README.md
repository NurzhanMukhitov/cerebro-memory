# Working Notes (Контур A)

Оперативные заметки, гипотезы, черновики конфигураций. Могут жить здесь или в сессии агента; между сессиями оперативный контекст может быть утрачен.

Стратегическая память (подтверждённые решения, цели, политики) хранится в Strategic Memory и в Decision Ledger; сюда не переносить без явного подтверждения пользователя.
